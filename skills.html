Skills

This page lists practical skills that help you work faster with the visualiser.

  • XML Import Skill: Load field XML from file or pasted text for each lidar.
  • Field Selection Skill: Choose a specific field from the parsed dropdown lists.
  • Monitoring Case Skill: Load monitoring-case XML and render selected cutoff path combinations.
  • Dual-Lidar Overlay Skill: Render Rear Left and Front Right fields together for comparison.
  • Measurement Skill: Use the distance tool to measure between two points on the canvas.
  • Coordinate Review Skill: Toggle coordinate labels to simplify or detail the view.
