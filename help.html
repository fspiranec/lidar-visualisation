Accepted files

Use XML exports from SICK fieldset tools for fields and monitoring cases.

Why is the render button disabled?

The button enables only after at least one lidar XML is parsed and one field is selected.

Why does nothing show after loading a case?

Check that matching field XML was loaded first and that selected cutoff paths are enabled.

How do I compare both lidars?

Load Rear Left and Front Right XML, select one field from each dropdown, then render together.

Need a clean view?

Turn off coordinate labels and use the clear measure button to remove measurement lines.
