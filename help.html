Supported inputs

  • Field XML: paste text or import .xml, .txt, .sdxml, .sdxlm.
  • Monitoring cases XML: paste text or import .xml, .txt, .casesxml, .sdxlm.

Render button is disabled

Load and parse at least one lidar field XML, then select one field in the corresponding dropdown.

Monitoring case controls are disabled

Monitoring controls activate after at least one valid cases XML is parsed (Rear Left and/or Front Right).

No geometry appears for a selected monitoring case

  • Confirm the matching field XML was loaded for the same lidar.
  • Check that at least one cut-off path checkbox (1-8) is selected.
  • Try rendering fields first to confirm the field parser works with your file export.

Coordinate and view tools

  • Show coordinates: toggles point labels.
  • Measure distance: click two points on the canvas to get distance and Δ values in mm.
  • Zoom in/out/reset: adjusts viewport quickly after render.
  • Clear measure: removes measurement overlays.

Editing and export

  • Edit mode: select points, edit X/Y values, and apply changes with undo history.
  • Export XML: exports Rear Left or Front Right XML with any applied point edits.

Navigation tips

Use Previous case, Next case, or enter a number in Case # and click Go to case.
