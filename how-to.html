How To Use This Tool

  1. Open the main page and paste or import Rear Left lidar field XML.
  2. Paste or import Front Right lidar field XML (optional for single-lidar view).
  3. Pick the desired field from each dropdown list.
  4. Click Render selected field(s) to draw polygons.
  5. Load monitoring-case XML for one or both lidars.
  6. Select a monitoring case and click Render monitoring case.
  7. Toggle cutoff path checkboxes (1-8) to filter displayed paths.
  8. Use Measure distance, then click two points on the map.
  9. Use Clear measure to reset measurement overlays.
