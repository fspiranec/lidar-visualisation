How To Use This Tool — v1.0.0 Official Release

Creator: Franjo Spiranec / Codex

1) Load lidar field XML files

  1. Load Rear Left XML using Paste XML or Import XML.
  2. Optionally load Front Right XML the same way.
  3. Select one field from each lidar dropdown.
  4. Click Render selected field(s).

2) Load monitoring cases and render by case number

  1. Paste/import monitoring cases XML for one or both lidars.
  2. Choose a value from Monitoring case list.
  3. Select one or more cut-off path checkboxes (1-8).
  4. Click Render monitoring case.

3) Navigate cases quickly

  • Previous case / Next case moves through loaded list.
  • Use Case # + Go to case for direct jumps.
  • The banner and badge display the currently rendered case (decimal + binary).

4) Work with the canvas

  • Measure distance: pick two points to show distance + Δx/Δy in mm.
  • Clear measure: remove measurement graphics.
  • Zoom in / Zoom out / Reset zoom: inspect dense polygons.
  • Show coordinates: toggle point labels for cleaner view.

5) Edit points and export XML

  1. Enable Edit mode.
  2. Select a point and modify Point X / Point Y.
  3. Click Apply point (undo supports recent edits).
  4. Export updated file using Export Rear Left XML or Export Front Right XML.
